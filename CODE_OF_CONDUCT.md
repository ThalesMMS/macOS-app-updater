# Code of Conduct

## Our standard

Please keep collaboration respectful, practical, and kind.

We want this repository to be a safe place for people to:

- ask questions without being mocked
- report bugs without being attacked
- suggest improvements without hostility
- disagree about technical decisions in a constructive way

## Unacceptable behavior

This includes harassment, personal attacks, hate speech, doxxing, deliberate intimidation, or sharing someone else's private information without permission.

## Enforcement

The maintainer may edit, hide, or remove comments, issues, pull requests, or other contributions that make the project less safe or less workable for others.

## Reporting

If you run into conduct problems, contact the maintainer privately through GitHub rather than escalating in public threads.
